8.18.  Show Scrollbars

Revision History
Revision $Revision: 2273 $ 2006-08-10 scb

The Show Scrollbars command enables and disables displaying the scrollbars. It may be useful to disable them if you are working in full-screen mode.

You can set the default for the scrollbars in the Image Window Appearance dialog.

8.18.1.  Activating the Command

  • You can access this command from the image menubar through View → Show Scrollbars.
