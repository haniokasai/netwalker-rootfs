10.33.  Color to Alpha

Revision History
Revision $Revision: 2326 $ 2006-08-03 scb

You can make colors of the active layer transparent by using the Color to Alpha command. You can find a description of this command in the section about the Section 11.36, “ Color to Alpha ” filter.

Note

This command operates only on the layer which is active at the time the command is called. Even after you choose another layer to be the active layer, the operation continues to affect the layer on which you performed the command.

10.33.1.  Activating the Command

  • You can access this command from the image menubar through Layer → Transparency → Color to Alpha.
