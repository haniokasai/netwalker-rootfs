8.16.  Show Menubar

Revision History
Revision $Revision: 2273 $ 2006-08-10 scb

The Show Menubar command enables and disables displaying the menubar. It may be useful to disable it if you are working in full-screen mode. If the menubar is not displayed, you can right-click on the image to access the menubar entries.

You can set the default for the menubar in the Image Window Appearance dialog.

8.16.1.  Activating the Command

  • You can access this command from the image menubar through View → Show Menubar.
