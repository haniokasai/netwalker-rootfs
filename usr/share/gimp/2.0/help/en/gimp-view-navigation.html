8.7.  Navigation Window

Revision History
Revision $Revision: 2273 $ 2006-08-10 scb

The Navigation Window command opens the navigation window. This allows you to easily navigate through the image, to set zoom levels and to move the visible parts of the image. You can find more information about using it in the Navigation dialog chapter.

8.7.1.  Activating the Command

  • You can access this command from the image menubar through View → Navigation Window,

  • or by using the keyboard shortcut Shift+Ctrl+N.

  • You can also access it more rapidly by clicking on the icon in the lower right corner of the image window.
