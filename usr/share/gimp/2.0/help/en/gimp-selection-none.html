7.3.  None

Revision History
Revision $Revision: 2424 $ 2006-08-15 scb

The None command cancels all selections in the image. If there are no selections, the command doesn't do anything. Floating selections are not affected.

7.3.1.  Activating the Command

  • You can access this command from the image menubar through Select → None. In addition, you can use the Selection Editor to access it.

  • You can also use the keyboard shortcut Shift+Ctrl+A.
