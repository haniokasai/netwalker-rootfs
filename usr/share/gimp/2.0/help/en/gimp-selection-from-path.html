7.7.  From Path

Revision History
Revision $Revision: 2424 $ 2006-08-15 scb

The From Path command transforms the current path into a selection. If the path is not closed, the command connects the two end points with a straight line. The original path is unchanged.

7.7.1.  Activating the Command

  • You can access this command from the image menubar through Select → From Path.

  • In addition, you can click on the Path to Selection button in the Path dialog to access the command.

  • You can also use the keyboard shortcut Shift+V.
