10.6.  Delete Layer

Revision History
Revision $Revision: 2419 $ 2007-12-04 j.h

The Delete Layer command deletes the current layer from the image.

10.6.1.  Activate the Command

  • You can access this command from the image menubar through Layer → Delete Layer, or from the local pop-up menu that you get by right-clicking on the Layer Dialog.
