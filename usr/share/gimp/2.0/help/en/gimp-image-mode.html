Figure 15.65.  The “Mode” submenu of the “Image” menu


The Mode submenu contains commands which let you change the color mode of the image. There are three modes.

9.3.1.  Activating the Submenu

  • You can access this submenu from the image menubar through Image → Mode.

9.3.2.  The Contents of the “Mode” Submenu

  • RGB

  • Grayscale

  • Indexed

  • Assign Color Profile (see Color management)

  • Convert to Color Profile (see Color management)
