10.34.  Semi-flatten

Revision History
Revision $Revision: 2272 $ 2007-12-02 j.h

The Semi-Flatten command is described in the Semi-flatten filter chapter. The command is useful when you need an anti-aliased image with indexed colors and transparency.

10.34.1.  Activate the Command

  • You can access this command from the image menubar through Layer → Transparency → Semi-flatten.
