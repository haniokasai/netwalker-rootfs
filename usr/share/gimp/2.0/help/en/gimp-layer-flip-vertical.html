10.42.  Flip Vertically

Revision History
Revision $Revision: 2272 $ 2006-08-17 scb

The Flip Vertically command reverses the active layer vertically, that is, from top to bottom. It leaves the dimensions of the layer and the pixel information unchanged.

10.42.1.  Activate the Command

  • You can access this command from the image menubar through Layer → Transform → Flip Vertically.
