11.38.1.  Overview

It identifies and modifies pixels which might cause problem when displayed onto PAL or NTSC TV screen.

11.38.2.  Activate the filter

You can find this filter in the image window menu under Colors → Hot.

11.38.3.  Options

Figure 15.202.  “Hot” filter options


Mode

You have to select the TV mode: PAL or NTSC.

Action

You can select:

  • Reduce Luminency

  • Reduce Saturation

  • Blacken: this will turn hot pixels to black.

Create a new layer

With this option, work will be performed on a new layer instead of the image. This will give you peace of mind!
