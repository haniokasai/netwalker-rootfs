5.6.  Open as Layer

Revision History
Revision $Revision: 2272 $ 2006-07-27 scb

The Open Image as layer dialog is identical to the Open Image dialog. With this dialog, you can open an image file, which is added to the current image as the top layer in the stack.

5.6.1.  Activate Command

  • You can access to this command from the image menubar through File → Open as layer,

  • or by using the keyboard shortcut Ctrl+Alt+O.
