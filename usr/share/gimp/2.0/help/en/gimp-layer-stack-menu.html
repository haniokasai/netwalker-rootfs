Figure 15.103.  The “Stack” submenu


The layer stack is simply the list of layers in the Layers dialog. The Stack submenu contains operations which either select a new layer as the active layer, or change the position of the active layer in the layer stack.

10.8.1.  Activate the Submenu

  • You can access this submenu from the image menubar through Layer → Stack.

10.8.2.  The Contents of the “Stack” Submenu

The Stack submenu contains the following commands:

  • Section 10.9, “ Select Previous Layer ”

  • Section 10.10, “ Select Next Layer ”

  • Section 10.11, “ Select Top Layer ”

  • Section 10.12, “ Select Bottom Layer ”

  • Section 10.13, “ Raise Layer ”

  • Section 10.14, “ Lower Layer ”

  • Section 10.15, “ Layer to Top ”

  • Section 10.16, “ Layer to Bottom ”

  • Section 10.17, “ The “Reverse Layer Order” command ”
