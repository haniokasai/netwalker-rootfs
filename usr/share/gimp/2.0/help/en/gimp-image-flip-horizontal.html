9.8.  Flip Horizontally; Flip Vertically

Revision History
Revision $Revision: 2281 $ 2006-08-07 scb

You can flip the image, or turn it over like a card, by using the Flip Horizontally or Flip Vertically commands. These commands work on the whole image. To flip a selection, use the Flip Tool. To flip a layer, use the functions of the Layer → Transform menu or the Flip Tool.

9.8.1.  Activate the Commands

  • You can access the horizontal flip command from the image menubar through Image → Transform → Flip Horizontally.

  • You can access the vertical flip command from the image menubar through Image → Transform → Flip Vertically.
