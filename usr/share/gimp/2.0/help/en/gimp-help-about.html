4.5.  About

The About command shows the About window, which displays information about the version of The GIMP you are running and the many authors who wrote it.

4.5.1.  Activating the About Command

  • You can access this command in the toolbox menu through Help → About

4.5.2.  Description of the dialog window

Figure 15.19.  The “About” dialog window
