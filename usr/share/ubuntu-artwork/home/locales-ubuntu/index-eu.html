Ongietorri Ubuntu 9.04-ra!

Ubuntu proiektua Ubuntu filosofiaren idealen inguruan dago osatua: softwareak doakoa izan beharko luke, software-tresnak bakoitzak bere hizkuntzan erabili ahal beharko lituzke, eta jendeak bere softwarea edozein modutara pertsonalizatu eta aldatu ahal beharko luke bere beharretara egokitzeko.

Ubunturekin laguntza lortzen:

Modu ugari daude Ubunturekin laguntza lortzeko.

"Sistema" menuan "Laguntza" hautatu ezazu Ubunturen gida eta dokumentaziora iristeko. Bestela, Ubuntu dokumentazio webgune ofizialera edo komunitateak eskainitako dokumentaziora ere jo dezakezu.

Nahiago baduzu gizakiekin hitz egin, hortxe dituzu posta-zerrenda, web foroak, edo Freenode IRC-zerbitzariko #ubuntu kanala.

Hala nahi baduzu, sostengu komertziala ere jaso dezakezu.

Ubuntun parte hartu ezazu:

Ubuntu komunitatean software-garatzaileak, dokumentazio-idazleak, itzultzaileak, artista grafikoak, eta Ubuntu egunero darabilen jendea topatuko duzu. Komunitate honetara batzera gonbidatzen zaitugu, Ubuntu zure beharrak hobe beteko dituen sistema eragile batean bihurtzen lagundu ahal dezazun.

Nola parte hartu jakiteko, bisita ezazu Ubuntu komunitatearen parte-hartze orria!
