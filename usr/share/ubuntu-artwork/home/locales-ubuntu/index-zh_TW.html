歡迎使用 Ubuntu 9.04!

Ubuntu 計劃建立於 Ubuntu 哲學思想(英) 之上︰ 軟體應可免費取得，軟體工具應讓使用者能以其當地語言操作使用，使用者應擁有客製及修改軟體以符合自身需求的自由。

從 Ubuntu 取得協助:

有很多方法可以從 Ubuntu 取得協助。

從系統選單中的"求助"項目可以取得 Ubuntu 指南及文件。您也可以從 官方文件網站(英)，社群建立文件(英) 以及 Ubuntu 正體中文 Wiki(中) 取得協助。

如果您較喜歡與人互動，您可以透過 Mailing Lists(英)，論壇(英)，Ubuntu 正體中文站(中) 取得協助，或者到 Freenode IRC 的聊天室頻道: #ubuntu(英)，#ubuntu-tw(中) 與社群成員們聊聊。

您也可以取得 商業支援(英)。

加入 Ubuntu 的行列:

Ubuntu 社群由軟體開發者，文件寫手，翻譯人員，圖像創作者，以及最重要的 - 每天使用接觸 Ubuntu 的人們所組成。我們邀請您加入此社群並協助將 Ubuntu 塑造成更符合您的需求的作業系統。

想知道如何加入 Ubuntu 社群，可以拜訪 Ubuntu 社群分享頁面(英) 或是 Ubuntu 正體中文社群(中) !
